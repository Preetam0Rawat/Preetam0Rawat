## 🚀 代表プロジェクト
### 日本語学習Webアプリケーション
React と Spring Boot を使用して開発したフルスタックの日本語学習Webアプリケーションです。
本アプリケーションでは、 2000以上の漢字と10,000以上の語彙を学習することができます。また、ユーザーは自分の学習進捗を記録・管理することができ、  
すべてのコンテンツはJLPTレベル（N5〜N1）に基づいて整理されています。
