## 👨‍💻 自己紹介

コンピュータサイエンス専攻のソフトウェアエンジニアです。  
JLPT N2 に合格しており、日本語での技術ドキュメントの読解および業務コミュニケーションが可能です。
現在は主にWeb開発を中心に学習および開発を行っています。
日本語能力とソフトウェア開発スキルを活かし、日本企業でエンジニアとして貢献することを目標としています。
---
## 🚀 代表プロジェクト
### 日本語学習Webアプリケーション

React と Spring Boot を使用して開発したフルスタックの日本語学習Webアプリケーションです。
本アプリケーションでは,2000以上の漢字と10,000以上の語彙 を学習することができます。

また、ユーザーは自分の学習進捗を記録・管理することができ、  
すべてのコンテンツはJLPTレベル（N5〜N1）に基づいて整理されています。

このプロジェクトは、企画から設計、開発、デプロイまで全て個人で行いました。
